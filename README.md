# 💬 Chat System

A real-time, high-performance chat application for instant communication.

## 🌐 Live Demo
Experience the chat in action here: [https://chat-system-c6t.pages.dev](https://chat-system-c6t.pages.dev)

## ✨ Features
- **Instant Messaging**: Real-time chat powered by WebSockets for zero-latency communication.
- **Presence Tracking**: See who is online and active in the room.
- **User-Friendly**: Simply enter your display name and join the conversation.
- **Developer Console**: Built-in tools for advanced room management and debugging.

## 🛠️ Built With
- **Frontend**: React & Vite
- **Backend**: Cloudflare Workers & Durable Objects
- **Storage**: Cloudflare KV
